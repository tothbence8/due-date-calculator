# Due Date Calculator
### Install

```shell script
yarn add due-date-calculator-tb
```
or
```shell script
npm install due-date-calculator-tb
```
### How to use?

```javascript
const calculate = require('due-date-calculator-tb');
const result = calculate('2021-03-25 11:10:10', 4);
```

### Run tests

```shell script
yarn && yarn test
```
